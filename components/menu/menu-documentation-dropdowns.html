Main settings

Name Type Description
Icons list Mixed Icons list is a feature of components. Complete documenteation here
Mega menu background image CSS To set a background image add class bg-menu and style background-image:url(image.jpg) to item object.
Example: <div class="mega-menu dropdown-menu multi-level row bg-menu" style="background-image:url(image.jpg)"> ... </div>
Mega menu full width CSS class To set full width mega menu add class mega-menu-fullwidth to navbar object.
Example: <div class="navbar navbar-default mega-menu-fullwidth" role="navigation"> ... </div>.

To set partial full width mega menu add class mega-menu-fullwidth to ul nav object.
Example: <ul class="nav navbar-nav mega-menu-fullwidth"> ... </ul>.

To remove full width to a single mega menu item add class dropdown to li menu item. Example: <ul class="nav navbar-nav mega-menu-fullwidth"><li class="dropdown"> ... </li></ul>.
